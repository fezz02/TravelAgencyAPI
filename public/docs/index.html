Introduction

An API that works with travels, tours and users

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Auth endpoints

POST Login

Login with the existing user.

Example request:
curl --request POST \
    "http://localhost/api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"ryan.susanna@example.net\",
    \"password\": \"\\\\jHS|JSEIC~7<i19\"
}"

const url = new URL(
    "http://localhost/api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "ryan.susanna@example.net",
    "password": "\\jHS|JSEIC~7<i19"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "access_token": "1|a9ZcYzIrLURVGx6Xe41HKj1CrNsxRxe4pLA2oISo"
}

Example response (422):


{
    "error": "The provided credentials are incorrect."
}

GET Travels

Returns paginated list of travels.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels?page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/travels"
);

const params = {
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):

GET Travel Tours

Returns paginated list of tours by travel slug.

Example request:

    --get "http://localhost/api/v1/travels/&quot;first-travel&quot;/tours" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"priceFrom\": \"\\\"123.45\\\"\",
    \"priceTo\": \"\\\"234.56\\\"\",
    \"dateFrom\": \"\\\"2023-06-01\\\"\",
    \"dateTo\": \"\\\"2023-07-01\\\"\",
    \"orderBy\": \"price\",
    \"orderDirection\": \"desc\",
    \"sortBy\": \"\\\"price\\\"\",
    \"sortOrder\": \"\\\"asc\\\" or \\\"desc\\\"\"
}"

const url = new URL(
    "http://localhost/api/v1/travels/&quot;first-travel&quot;/tours"

const headers = {
